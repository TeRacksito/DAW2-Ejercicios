Elementos de incrustar contenido

img

El elemento <img> se utiliza para incrustar imágenes en un documento HTML.

iframe

El elemento <iframe> se utiliza para incrustar otra página HTML dentro de la página actual.

embed

El elemento <embed> se utiliza para incrustar contenido externo, como archivos multimedia o documentos.

object

El elemento <object> se utiliza para incrustar objetos multimedia, como videos, audio o documentos PDF.

param

El elemento <param> se utiliza para definir parámetros para plugins incrustados con el elemento <object>.

video

El elemento <video> se utiliza para incrustar videos en un documento HTML.

audio

El elemento <audio> se utiliza para incrustar archivos de audio en un documento HTML.

source

El elemento <source> se utiliza para especificar múltiples fuentes para elementos multimedia como <video> y <audio>.

track

El elemento <track> se utiliza para especificar pistas de texto para elementos multimedia como <video> y <audio>.

canvas

El elemento <canvas> se utiliza para dibujar gráficos mediante scripts (normalmente JavaScript).

map

El elemento <map> se utiliza para definir un mapa de imagen, con áreas clicables.

area

El elemento <area> se utiliza dentro de un <map> para definir áreas clicables en una imagen.

svg

El elemento <svg> se utiliza para definir gráficos vectoriales escalables.
